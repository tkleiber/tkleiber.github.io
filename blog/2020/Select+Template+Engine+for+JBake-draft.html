Select Template Engine for JBake

Posted by Torsten Kleiber on July 17, 2020
Tags: JBake Blog

JBake supports several Template Engines. As I want to change the delivered templates, I want to decide, which template engine is most suitable for me.

Freemarker

  • Transferred to Apache

  • not more the yearly releases

  • IntelliJ Support

Groovy

  • no IntelliJ Support

SimpleTemplateEngine

  • "for basic templates"

XmlTemplateEngine

  • "works well when the template and output are valid XML"

MarkupTemplateEngine

  • a very complete, optimized, template engine

Thymeleaf

  • last stable version 3.0.11 from 29 October 2018

  • IntelliJ Thymeleaf plugin: Download

Jade

  • based on npm

  • wrapped via Jade4J?

  • IntelliJ Pug (Jade) Template Engine
